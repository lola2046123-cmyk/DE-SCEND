> **版本**：与 `config/game-events.json._meta.version = 0.05` 对齐。

## 21. 已知文案分叉（供后续统一时参考）

1. **配额未达成**：`copy.quotaNudge` 与 `showDebtWarning` 默认正文均为局内划扣口径；若配置注入更长模板，宜与许可脚注「信用折偿」一致。
2. **叙事双轨**：引擎 `getOutcomeNarrative`（B）与场景 `formatOutcomeNarrativeHTML`（A）主题相近、句式不同。
3. **对外品牌**：主标题「暴富」仅用于启动/舞台品牌区；局内术语仍以配额、清算、归档为主。
4. **0.04 期"权力制衡 vs 身份卡"分叉已闭合**：UI 与产品文档统一为「封存货舱选盘」（HTML `#locker-selection-overlay` + 文档 §5.5）；旧"`选一张身份卡`"文案已不再出现。
5. **背景音控件**：0.05 起 `aria-label` 以「背景音：开启 / 关闭」为法律与读屏锚点，面向用户文档可继续使用"背景音开关"口径，避免再混入"沉浸模式"。
6. **5 身份标签源**：0.05 起 `IDENTITY_LABELS` 与 `PASSENGER_COMMITTEE` 已在 HTML 内联补齐 5 身份；与配置 `passenger.types.*.label` 形成档案口径 / 简称双轨。后续统稿时建议把审查菜单与立绘标签收敛到 `passenger.types[id].label` 单源（HTML 侧自动叠加 `// ` 前缀），避免再加新身份时双轨漂移。
