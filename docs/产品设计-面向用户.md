# 暴富电梯 · 探测员手册

> **联邦控制局 · 计价规程 · 版本 0.07.3**
> 适用对象：已授权探测员 / 首次登梯申请人
> **本期修订要点**：House Edge 第二阶段已落地——**信用评级**（跨局累计破产影响下一趟权益）、**浮盈折现**（账面越肥局方抽水越重）、**撤离税**（超额盈余撤离须缴 dividend tax）；委员会新增三条人格化触发（大赢低语 / 连胜探测协议 / 高偏移加码）；货舱清点心流优化（等待压缩 ~61% + 阅读期可点击跳过）；数值颜色全局对齐**中国金融市场惯例**（红涨绿跌）。
> **货舱物品全面本土化**：飞天茅台 / 安宫牛黄丸 / 户口本 / 1980 猴票 / 蝴蝶缝纫机 / 永久二八大杠 / BB 机 / P2P 暴雷合同 / 共享单车押金条……FBC × 中国梦的三段年代切片。

## 前景提要

没有人知道这栋楼有多少层。

联邦控制局的测量记录在第 31 层终止，原因标注为"设备失联"。此后派去的探测员陆续回来，带着一个共同描述：**电梯还在往上走**。

局方在第 3 层设立了临时计价事务所。规程很简单——你带一笔授权资产上去，回来的时候把增值部分归档，局方抽成，剩下的是你的。问题是，每层的货舱里封存着什么，只有打开才知道。

这就是他们管它叫**暴富电梯**的原因。

也是他们每隔一段时间就要发一批新探测员上去的原因。

---

你扮演**一名探测员**，持有联邦控制局开具的「资产上行计价许可」，驾驶轿厢向上逐层开仓结算。规程赋予你两项核心权利：

- **选择**：在每层开门后，亲手从 2～4 只封存货舱中选一只开启；
- **撤离**：在任意决策窗口，带走当前账面净值并终止本趟上行。

局方在另一边观察着。每当你犹豫过长、偏移指数攀升到临界，他们就会介入——以委员会的名义。

**关门。上行。开仓。选择还是等死。**

## 一、登梯前：授权许可

首次进入游戏或每趟结束后，你会看到 **「联邦资产上行计价许可」** 面板。

**授权金额**是你这趟带上去的启动资本，范围 **¥10 ～ ¥99,999**。也可以直接点四档快捷金额（¥100 / ¥200 / ¥500 / ¥1,000）。

面板会显示本趟的**配额线**——这是局方按你的**授权金额自动核定**的最低回收目标。授权越小，配额倍率越高（最高 5×），越像"以小博大"；重仓 buy-in 倍率收敛（最低 2.5×），越接近"稳健入场"。

| 授权金额 | 配额倍率 | 档位 |
|---|---|---|
| ¥10 ～ ¥250 | 5.0× | 小本博弈 |
| ¥251 ～ ¥600 | 4.0× | 标准入场 |
| ¥601 ～ ¥1,500 | 3.0× | 杠杆扩张 |
| ≥ ¥1,500 | 2.5× | 高净值 |

> 例：买入 ¥200 → 配额 ¥1,000；买入 ¥1,000 → 配额 ¥3,000；买入 ¥5,000 → 配额 ¥12,500。

越过配额线再撤离，超额部分 100% 归你；没过线就强行撤离，差额按「失信协议」归档为债务。

**配额过线即奖励触发**：HUD 数字一次性渐染光晕，stage 顶部闪现 `// FBC §3.1 — 配额已达成 ×<倍率>`，**下一次选舱倒计时 +1 秒**——这是局方默许的喘息。

### 信用评级：你在局方账本里的位置

每趟结束后，局方会根据你的**生涯破产记录**更新你的信用档案。下一趟登梯时，面板顶部会显示当前评级：

| 评级 | 状态 | 含义 |
|---|---|---|
| **AAA** | 联邦优先通道激活 | 连续无破产 + 净盈利，局方流通损耗略减 |
| **A** | 标准档（默认） | 正常规程，无加成无惩罚 |
| **B** | 联邦侧偿区间已激活 | 累计 1–2 次破产，偏移压力微增 |
| **C** | 流动性管理已介入 | 累计 3–4 次破产，撤离额外扣除 |
| **D** | 债务重组期 | 累计 ≥5 次破产，初始杠杆上限降低，面板脉冲警告 |

> 评级 A 时不显示任何提示。只有当委员会需要提醒你"已有记录"时才会出字。

> **战绩菜单**（右上角 ≡）随时可以查规则说明和你的历史战绩。背景音开关也在右上角——关闭只静音背景音乐，货舱开启音、倒计时心跳、FBC 盖章音等事件音效**始终保留**。

确认金额后，点 **「启动电梯」**，关门，上行开始。

## 二、核心循环：上行 · 选仓 · 撤或继续

每趟上行都在重复同一个回路，直到你撤离或被清算：

```
关门上行 → 随机跳 1～8 层 → 货舱弹出 → 限时选一只开启
→ 货舱清点（三件子产物逐件揭露） → 账面变动
→ 决策：继续上行 / 风险撤离
```

### 上行与楼层跳跃

按下红色▲主钮（桌面键盘亦可用 Space 调出指令圆盘 → 上行），轿厢关门，**随机向上跳 1～8 层**。每跳一层都会按"基线增长"重算账面，但乘数会随**资产 / 配额比值**逐步收紧：

| 资产 / 配额 | 每层基线 | 局方备注 |
|---|---|---|
| < 1.5× 配额 | **+10%** | 养肥期 — 玩家上头不被打扰 |
| 1.5 ～ 3× 配额 | +6% | 节流期 — 局方开始观察你 |
| 3 ～ 6× 配额 | +3% | 冷冻期 — 涨幅几乎要被流通损耗吃光 |
| ≥ 6× 配额 | +1.5% | 守钱比赚钱难 — 落袋为安信号 |

**守住比赚到更难**——这是规程对"暴富心理"的官方逆向回应。早期养肥让你上头，越接近肥尾、每层涨幅越被收紧。

偏移指数也随每一层累积——跳得越多，压力越大。

### 封存货舱：这一层你开哪只

到站后，屏幕中央弹出 **2～4 只封存货舱**，每只标着 `LOT-XX` 编号。

**关键：每只货舱在弹出之前就已经预先"封装"了独立的结算结果。** 你选哪只，就是哪只的命运——其余的结果随浮层关闭永远消失。

屏幕底部有一条**倒计时进度条**，时长由柜数决定（每柜约 4 秒，总区间 6～16 秒）。**你必须在倒计时结束前点击一只货舱开启。**

倒计时剩余 30% 时，进度条变红并出现心跳提示音——这是局方给你的最后催促。

> 你有的不只是感觉。如果车厢里有 **情报员** 或 **清扫员**，他们可能在货舱弹出前小声给你一句话。信还是不信，是你的判断。

## 三、FBC 代决议：等于放弃

倒计时归零，但你什么都没选——**委员会会替你做决定**。

这不是"稳妥的等待"。这是规程对你怠惰的惩罚：

| 代决议效果 | 说明 |
|---|---|
| 随机开一只柜 | 系统随机高亮一只货舱（红框闪烁 + 盖章音） |
| **账面折扣 62%** | 本层净增益只保留 62%；净亏损则放大 138% |
| **偏移指数 +0.45** | 这是单步累积里最重的一档 |
| 档案记录 | 你的生涯档案里 `代决议次数 +1`，委员会留了案底 |

委员会横幅会闪出一行字：`FBC · 委员会代决议 · 强制开启 LOT-XX`。

**你在窗口期没有行使裁量权——局方按规程接管，你为犹豫付出账面与偏移的双重代价。**

## 四、货舱清点：数字背后的三件事

选定货舱后，本层结算不会直接给你一个数字。

结果会被拆成 **3 件子产物**，像翻账册一样逐件揭露：名称、档案号、金额变动，一条条出来。最后一件通常是最大的。

**颜色即方向**：<span style="color:#ff6b6b">**红光 = 正向（收益）**</span>，<span style="color:#4ade80">**绿光 = 损失**</span>，灰白 = 中性。最末一张的颜色和金额就是本层结算的定论。

| 结果类型 | 三件节奏 | 体验感 |
|---|---|---|
| **正面（POSITIVE）** | 25% / 35% / 40% | 越翻越值，末件最大 |
| **超现实双倍（DOUBLE）** | 20% / 30% / 50% | 节奏陡，第三件是真正的裂变红利 |
| **负面（NEGATIVE）** | 30% / 30% / 40% | 三笔全是损失，逐件让你皱眉 |
| **强制清算（LIQUIDATION）** | — | 跳过清点，直接进入清算结果面板 |

> **熟悉规程的探测员**：卡片停留阅读时，**点击空白区域**即可跳至下一张，无需等倒计时。末张点击将直接进入决策状态。

子产物的名称随楼层区域变化——

- **行政区（1～10 层）**：飞天茅台、户口本、学区房份额、1980 猴票、永久二八大杠……局方眼里的"硬通货"切片。
- **维护区（11～25 层）**：蝴蝶缝纫机、BB 机、过期急救箱、断丝铜联轴节、锈蚀销栓……国营老三件的副本残档。
- **超自然区（26+ 层）**：安宫牛黄丸、田黄石、模因护符、诅咒账册、P2P 暴雷合同、共享单车押金条……连局方都不愿背书的"另册"。

这些是局方留给你的叙事残片，看清楚再走——后期会有"实物图档"补充。

### FBC §4.7 — 联邦资产流通损耗

三件子产物揭露完毕、账面变动出现的同一帧，三位一体结果帧底部会浮出一行小字：

```
[ A-007 三联印章 ] | +¥320
// FBC §4.7 已扣 ¥13 · 4% 流通损耗
```

这是局方对**正向收益**抽取的"联邦资产流通损耗"——按结果带分档：

| 结果带 | 流通损耗 | 局方说法 |
|---|---|---|
| 普通正面（POSITIVE） | **4%** | 流通基础税 |
| 超现实双倍（DOUBLE） | **8%** | 大赢大税 |
| 金板正面（GOLDEN） | **12%** | 肥尾另算 |
| 负面（NEGATIVE） | 0% | 输了不再加税 |
| 强制清算（LIQUIDATION） | 0% | 局方按债务程序处理 |

**你赢得越多 / 越频繁，FBC 切走的越多**——这不是惩罚，这是**计价规程**。配合上一节的基线分段衰减，赢钱速度被挤出"线性"，真实 RTP（玩家回报率）从 0.05 阶段的 127% 压回约 117% 区间。

> 注脚仅在正向收益且税额 > 0 时显示。负面 / 清算不会出现这一行。

### FBC §5.2 — 浮盈折现（Surplus Discount）

**流通损耗还不够**。当你的账面已经大幅超出配额，局方会在 §4.7 之前额外扣一道"浮盈折现"——资产越肥，折现越重：

| 当前资产 / 配额 | 折现乘子 | 局方备注 |
|---|---|---|
| < 1.5× 配额 | 无折现 | 规程内 |
| 1.5 ～ 3× 配额 | ×0.95 | 轻度干预 |
| 3 ～ 6× 配额 | ×0.88 | 重度干预 |
| ≥ 6× 配额 | ×0.80 | 极端干预 |

结果帧同样会显示一行注脚：

```
// FBC §5.2 浮盈折现 ×0.88 · 已扣 ¥28
```

> 这两道扣法是叠加的：先折现，再流通损耗。账面越大，离开越合算。

## 五、偏移指数：你的临界值

偏移指数是**本趟的累积风险压力**，显示在 HUD 右侧的红色条。

它越高，**坠毁（资产清零）的概率越大**，所有结果带的危险边界都在向坏处偏移。

| 触发项 | 偏移增量 |
|---|---|
| 每跳一层 | +0.08 |
| 负面结果 | +0.12 |
| 超现实双倍 | +0.35（双倍也有代价） |
| FBC 代决议 | **+0.45**（最重） |
| 正面结果 | +0.05（小幅）|
| 偏移涨潮（环境事件） | 当轮偏移增速 ×2 |

**HUD 文字分档：** 稳定（<20%）→ 波动（<42%）→ 偏高（<65%）→ **临界**（≥65%）

> 偏移满档（300%）= 本趟清算，无条件。

## 六、乘客：同梯的陌生人

偏移越高，越到高楼层，遇到乘客的机会也越大。最多同时有 **2 名乘客**。

乘客分五种身份，但**你看不出来是谁**——除非你消耗一次软盘模块去「审查」。

### 五种身份

| 身份 | 表面称呼 | 对你的影响 |
|---|---|---|
| **监理员 VIP** | 审计在场 | 降低坠毁/负面概率，资本 ×1.15；被审查后可降低偏移 |
| **红圈商人 SCAMMER** | 观测员入场 | 提高坠毁/负面概率，资本 ×0.85；**100% 伪装**，审查揭穿后逃离 |
| **希斯潜伏者 DANGER** | 共振偏高 | 明显提高坠毁概率；**50% 伪装**；揭穿后会让偏移上涨，且每层结算结果可能再随机放大或缩小 |
| **内线情报员 INFORMER** | 线人入场 | **不影响阈值**；有 85% 概率在选盘前给你一条情报，准确率 78% |
| **清扫员 CLEANER** | 清扫员到位 | **微降清算/负面概率**；有 45% 概率给情报，准确率只有 55%（更容易是假情报） |

> **乘客上车时**，委员会会播报一条简短口令。口令不会直接告诉你是谁，但语气里有线索。

### 情报员的那句话

如果车上有未伪装的情报员或清扫员，在货舱弹出前可能出现一句话，例如：

> *「情报员低声: "03 号有油水。"」*
>
> *「清扫员摇头: "07 号？我刚擦过血迹。」*

**INFORMER 有 78% 的概率说真话；CLEANER 只有 55%。** 两者都可能在同一趟中撒谎。把情报当做概率参考，而不是结论。

### 审查：揭露伪装

**点击乘客卡片 → 执行审查**（桌面键盘亦可 Space 调圆盘 → 执行审查），消耗 1 次软盘次数，优先揭露仍在伪装的那一位：

- 揭穿 **SCAMMER** → 他逃离，威胁解除
- 揭穿 **DANGER** → 偏移 +0.17，希斯相关叙事触发
- 揭露 **VIP** 真身 → 偏移 -0.11，审计背书

## 七、风险撤离：拿走已有的

在任何**决策窗口**（货舱已结算、无入侵状态），你都可以按绿色▼主钮撤离。

当前账面整数部分**全部带走**，本趟以「成功撤离」结束，档案封存。

### FBC §6.1 — 撤离税（Evacuation Tax）

如果撤离时你的账面**超过配额线**，超额部分将被征收**撤离税**：

```
超额 = 当前账面 − 配额
撤离税 = 超额 × 12%
实际到手 = 当前账面 − 撤离税
```

结算面板会单独显示一行：

```
撤离税（超额 ×12%）  −¥420
```

> 配额线以内的部分不征税。账面刚过配额一点点、或者账面很低的情况下，撤离税影响有限；但账面是配额的 3× 以上时，这道税会实质性压低到手金额——这是浮盈折现的出口端配套。

### 委员会对你的关注

委员会现在会主动"出声"——三种情形触发他们的低语：

| 触发情形 | 委员会行为 |
|---|---|
| **单层大赢**（delta > 配额 × 4） | 委员会随机播出一条低语台词（叙事警示） |
| **连续正向结算**（探测协议激活） | 面板顶部显示探测协议行；局方悄悄把每层流通损耗加码 +2% | 
| **偏移指数 ≥ 70%**（高偏移加码） | 选仓倒计时被压缩 10%；选盘副标题切换为「高偏移区 · 系统已介入」 |

> **连胜触发质询**：连续 2 层正面/双倍结算后，下一次撤离有 30% 概率先弹出委员会质询——「你真的要走？」你可以选「继续上行」或「步出电梯」。他们不强迫你，但他们在看。

## 八、Hiss 入侵：五秒窗口

有时候，门刚关上，屏幕开始变形——**Hiss 裂变入侵**触发。

一个全屏倒计时出现：**5 秒**。

- 倒计时内你仍可以 **风险撤离**（绿色 ▼ 主钮；桌面键盘亦可 ⌘R）带着现有账面离开；
- 上行在此期间锁定；
- **5 秒到了什么都没做 → 强制清算，资产归零。**

Hiss 入侵在楼层 ≥5 时才可能触发，触发概率随乘客身份变化：**DANGER** 在场显著升高，**VIP** 在场有抑制效果。

## 九、应急制动：买一次缓冲

软盘模块（初始 3 次）除了审查，还有另一个用途：**武装应急制动**。

在决策状态下，**点击轿厢右侧拉杆**（偏移 >20% 时出现，桌面键盘亦可 Space → 圆盘 → 应急制动），制动被武装——此后下一次强制清算时：

**资产不归零，而是保留本层基线快照的约 30%。**

> 示例：基准账面 ¥200，触发清算时留存约 ¥60，而不是 ¥0。

制动只触发一次，触发后消耗，需要重新武装。

## 十、楼层区域：三段式难度

整趟上行分为三个区域，UI 风格、音效、以及每层弹出的货舱数量都会随之变化：

| 楼层 | 区域 | 货舱数 | 感受 |
|---|---|---|---|
| **1 ～ 10** | 行政区 | 2 只 | 秩序感，相对平稳 |
| **11 ～ 25** | 维护区 | 3 只 | 灯光不稳，压迫加强 |
| **26 +** | 超自然区 | 4 只 | 电梯在颤，规律失灵 |

> 前 3 层是养猪期，强制 2 只货舱，让你先感受一下节奏。

每 5 层（5、10、15…）还可能触发一次**环境事件**，持续到下一次被覆盖：

| 事件 | 效果 |
|---|---|
| **灯光熄灭** | 叙事/视觉扰动，不改变概率 |
| **偏移涨潮** | 本轮偏移累积速度 ×2 |
| **阈值漂移** | 坠毁带上限额外扩张 |

## 十一、生涯档案：你的信用记录（博彩对账三件套）

每趟结束后，结果会写入本机存储的**生涯档案**。战绩页（右上角 ≡ → 生涯档案）按博彩流水账格式 5 行展示：

| 行 | 字段 | 含义 |
|---|---|---|
| 1 | **累计建仓投入** / **累计核销资产** | 历次 deploy 投入的本金总额 vs. 历次成功撤离到账的总额 |
| 2 | **综合损益结算 · NET P&L** | `核销资产 − 建仓投入` 的真实净盈亏（**正红 / 负绿 / 平账灰**） |
| 3 | **历史最高深度** / **累计上行楼层** | 去过的最高楼层 + 累计跳层总数 |
| 4 | **破产记录** | 被清算归零的次数 + 累计损失（≥3 次红色高亮） |
| 5 | **FBC 委员会代决议次数** | 你在倒计时窗口没有行使选择权的累计次数 |

> **颜色惯例**：本作所有财务数值遵循**中国金融市场惯例**——红色 = 正向（涨 / 收益），绿色 = 负向（跌 / 亏损）。货舱清点卡片、HUD 净资产变动、交易收据、NET P&L 均使用同一套用色。

> **关键设计**：v0.06 起，"债务撤离"也会写入 `累计核销资产`（旧版只记建仓出、不记债务撤离回流，导致 NET P&L 永远偏负）。`配额过线次数 / 债务撤离次数 / 破产次数` 也在系统内累加，作为局方**信用评级**的背景字段——你的下一趟权益由此决定。

委员会在读你的档案。累计清算 ≥3 次，他们的评语是：*「委员会对您的稳定性表示担忧。」* 累计核销 ≥¥5,000，他们改口：*「您正在重塑局内的财务结构。」*

> 档案存于本机浏览器，清除网站数据会重置所有记录。右上角战绩菜单可随时查看，也可以在那里手动清档。

## 十二、操作速查

> 本作以**移动端触屏**为主，所有核心操作都通过 HUD 上的物理按钮 / 物理隐喻完成；桌面键盘快捷键仅作辅助增强。

### 主入口（移动端 + 桌面通用）

| 操作 | 主入口 |
|---|---|
| 上行 | HUD 底部红色 **▲ 主钮** |
| 风险撤离 | HUD 底部绿色 **▼ 主钮** |
| 执行审查 | **点击乘客卡片** → 弹出菜单 → "执行审查" |
| 武装应急制动 | **轿厢右侧拉杆**（偏移 > 20% 时出现） |
| 选择货舱 | **点击柜门** |
| 查阅资产异动档案 | 决策态下点击三位一体帧底部 `📄 查阅异动档案` |
| 查看战绩 & 规则 | 右上角 **≡ 菜单** |
| 关闭弹窗 | 弹窗自带按钮 / 点击半透明遮罩 |

### 桌面键盘增强（仅键鼠玩家）

桌面端额外提供 `Space` 键调出**指令圆盘**——这是把上面四件事（上行 / 撤离 / 审查 / 应急制动）集中到一个面板里的快捷集合，**没有独占功能**，移动端不需要也不暴露此按钮。

| 操作 | 桌面快捷键 |
|---|---|
| 调出 / 关闭指令圆盘 | `Space` |
| 风险撤离 | `⌘R` / `Ctrl+R` |
| 选择已聚焦的货舱 | `Enter` 或 `Space` |
| 关闭圆盘 / 弹窗 | `Escape` |

## 十三、几条经验

**关于代决议：** 倒计时还剩一秒，随便选一只也比等着委员会来好。62% 折扣和 +0.45 偏移是你主动放弃的代价，不是随机事件。

**关于情报：** INFORMER 78% 真，CLEANER 55% 真——如果 CLEANER 说"07 号别碰"，你跑去选 07 也不是没理由的。

**关于双倍（DOUBLE）：** 账面翻倍，但偏移 +0.35，是代价最高的好结果。连续两次双倍后撤离，往往是正确选择。

**关于偏移临界：** 偏移 ≥65% 进入「临界」状态，此时 Hiss 入侵概率、强制清算带宽双双扩大。如果账面已经够了，「临界」就是撤离信号。

**关于应急制动：** 它不能无限复用。武装一次、用一次。在偏移很高、账面还不错时，提前武装是保命选项。

**关于 FBC 流通损耗 + 浮盈折现：** 两道扣法叠加。养肥期（资产未达配额）每层 +10% 还有"利润空间"；到了"冷冻期"（资产 ≥3× 配额），每层涨幅几乎要被流通损耗和折现双重吃光——继续上行的边际收益在递减。**账面越肥，离开越合算。守住比赚到更难，落袋为安从来不丢人。**

**关于撤离税：** 超额越多，撤离税的绝对金额越大——但这是**出口税**，不是持有税。如果账面已经超出配额很多，算一下税后到手再决定要不要继续上行。配额刚过线一点点时，撤离税几乎可以忽略不计。

**关于信用评级：** 破产不只清算这一局——它会拉低你下一趟的评级，让委员会对你的流通损耗和偏移施加额外压力。偶尔破产影响有限，但连续破产会真实影响权益。信用评级 D 时登梯，你面对的已经是更难的条件。

**关于配额过线：** 过线即奖励——HUD 数字渐染光晕、stage 顶部闪现 `// FBC §3.1 — 配额已达成`，下一次选舱倒计时 +1 秒。这不是装饰，这是局方默许的喘息——用好这一秒。

**关于清点跳过：** 货舱清点时如果你对结果一目了然，直接点击空白区域跳过卡片——整个清点可以在几秒内完成。但末张一定要看清楚：那才是本层真正的账面变动。
